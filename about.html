About EduTrack

EduTrack is a web-based platform designed to simplify student record keeping. It helps users to create, manage, and analyze student data effectively using built-in features like search, sort, course statistics, and dynamic UI updates.

Developed using HTML, CSS, and JavaScript with LocalStorage, EduTrack is a professional-level academic tool that blends design, functionality, and interactivity for the modern educator.
